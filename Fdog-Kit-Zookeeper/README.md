使用zookeeper-c库编写的zk可视化操作工具

支持：
1. 连接信息缓存
2. 同时连接多个zk
3. 支持实时更新当前选中节点数据，如果取消勾选实时更新，则有绿色圆点进行提示
4. 支持快速搜索，支持大小写
5. 支持json数据格式化，节点url解码
6. 支持增删改查
7. 新增编辑区，可用于临时编辑处理
